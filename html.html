시맨틱 요소

시맨틱 웹이 중시되어서 여러 시맨틱 요소가 HTML5 에서 새롭게 만들어졌습니다.
 : 의미가 있는, 의미론적인

시맨틱 요소 사용해야 하는 이유

 엔진이 시맨틱 요소를 더 선호, 구글, 네이버 등에서 상위 노출
여러 개발자가 작업할 때 편리

시맨틱 요소의 종류

 : 페이지의 가장 아랫부분에 위치하며 라이선스, 주소 연락처 등을 넣습니다.
 : 페이지나 해당 섹션의 가장 윗부분, 사이트의 제목이 보통들어갑니다.
 : 상단바 등 사이트를 안내하는 요소로 사용. 보통 <ul> 을 넣어 목록 형태로 사용
 : 문서의 주된 콘텐츠를 표시합니다.

HTML의 속성(attributes)

 : 고유한(unique) 이름을 붙일때, 중복 불가, Selector 기호 : #
 : 반복되는 영역을 유형별로 구분할 때, 중복 가능, Selector 기호 : .
