Flexbox

박스를 유연하게 늘리거나 줄여 레이아웃을 잡는 방법

display:

부모 박스 요소에 적용해 자식 박스의 방향과 크기를 결정하는 레이아웃 구성법

부모 요소에 적용해야하는 Flexbox 속성

 : 정렬 축 정하기, 가본값은 가로 정렬, (row, column, ...)
